Chart embed instructions

Embed charts from our Statistical bulletins, articles and compendia in your own website. Please not this functionality is still at an early stage of development. We will be looking at making this simpler and clearer from the content in the near future.

Using an iframe


<iframe src="https://www.ons.gov.uk/embed?uri=/economy/inflationandpriceindices/bulletins/consumerpriceinflation/sept2016/cc765147&notes=false&source=false&subtitle=false&title=false" width= "475" height= "356" frameborder="0"></iframe>

Using Pym


<div id="chart1"></div>
                  

<script type="text/javascript" src="https://pym.nprapps.org/pym.v1.min.js"></script>
                  

<script>
var pymParent = new pym.Parent('chart1', "https://www.ons.gov.uk/embed?uri=/economy/inflationandpriceindices/bulletins/consumerpriceinflation/sept2016/cc765147", {});
</script>

Available true/false flags

The following flags are available to customise the display of the chart and follow the format "&notes=false". By default all are set to "true".
  • title
  • subtitle
  • source
  • notes
